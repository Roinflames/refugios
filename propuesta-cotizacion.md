# Propuesta y Cotizacion - Sistema de Gestion para Cabanas

## Cliente
GD

## Valor del proyecto
- Total: CLP $450.000 + IVA (Factura)
- Pago 1: 50% al inicio (CLP $225.000 + IVA)
- Pago 2: 50% al entregar MVP (CLP $225.000 + IVA)

## Alcance MVP (2-3 semanas)
- Gestion de ventas, gastos y reservas.
- Registro de datos de huespedes.
- Emision de boletas y facturas.
- Clasificacion de reserva por origen (sitio web, Airbnb, Booking, otros).
- Registro de forma de pago.
- Migracion inicial de datos historicos (anio anterior), desde ContaVirtual/Excel/CSV.

## Entregables
1. Plataforma web operativa para uso diario.
2. Panel con modulos de ventas, gastos, reservas y huespedes.
3. Reporte base mensual por movimientos.
4. Carga inicial de historicos.

## Supuestos
- El cliente entrega datos historicos en formato exportable.
- Ajustes mayores de alcance se cotizan por separado.

## Inicio
- Se inicia al confirmar propuesta y recibir abono del 50%.
